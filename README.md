# devops-assignment

## Useful commands Commands:
* Create VM using Vagrant:
```bash
# Assuming you are running from the VagrantFile dir:
vagrant up
```

* Delete VM using Vagrant:
```bash
# Assuming you are running from the VagrantFile dir:
vagrant destroy
```

* SSH into the Vagrant VM using ssh:
```bash
# Assuming you are running from the VagrantFile dir:
vagrant ssh
```
